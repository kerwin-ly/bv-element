Based on [bv-element](http://element.eleme.io/#/en-US/component/quickstart) and enhanced for some business requirements.
## Env

* node: `v11.15.0`

## Quick Start

Install Dependencies

```shell
npm run bootstrap
```

Run Project
```shell
npm run dev:play
```
